#php get 

使用GET來傳送資料

GET就是常見的網址後面會帶一串參數

例如:http://127.0.0.1/index.php?str=string

將參數string儲存給變數str

使用的方法有兩種一種是直接轉網址的時候就把參數帶上

另外一種則是參考範例

|類型                                        |說明                                         |
|--------------------------------------------|---------------------------------------------|

|[send](transfer/send.php)                   |php使用get來傳送資料                         |
|[receive](transfer/receive.php)             |php使用get來接收資料                         |
